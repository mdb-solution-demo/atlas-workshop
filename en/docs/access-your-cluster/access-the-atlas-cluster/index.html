1. 访问您的集群

我们有多种方式连接到MongoDB Atlas集群，除了应用驱动连接外，常见的有以下三种方式，大家可以分别尝试一下

  • 1. Data Explorer
  • 2. Compass
  • 3. MongoDB Shell

数据浏览器是默认启用的，可通过集群视图上的Collections选项卡访问。

编辑/添加文档​

请参阅以下文档，了解如何“修改”，“创建”和“删除”文档。

作为练习，您可以在test数据库下创建一个名为users的集合：

在插入文档选项卡中插入如下用户文档数组。

[
  {
    "name": "Alice",
    "age": 26,
    "email": "alice@abc.com"
  },
  {
    "name": "Bob",
    "age": 43,
    "email": "bob@def.com"
  },
  {
    "name": "Carol",
    "age": 19,
    "email": "carol@xyz.com"
  }
]

安装 Compass​

选择您的平台

Compass on MacOSCompass on MacOS (M1) Compass on Windows Compass on Ubuntu

请按照以下指南在您的工作站操作系统上安装与Compass相关的二进制文件。

使用 Compass​

打开Compass后，您将获得连接字符串输入屏幕。

使用Atlas“连接”选项卡获取您访问Atlas部署所需的字符串。

复制连接字符串。

通过Compass进行粘贴和连接：

请参阅以下文档 ，了解如何“修改”，“创建”和“删除”文档。

作为练习，您可以在test数据库下创建一个名为users的集合：

在“ADD DATA”>“Insert document” json视图中插入一个用户文档数组。

[
  {
    "name": "Alice",
    "age": 26,
    "email": "alice@abc.com"
  },
  {
    "name": "Bob",
    "age": 43,
    "email": "bob@def.com"
  },
  {
    "name": "Carol",
    "age": 19,
    "email": "carol@xyz.com"
  }
]

安装 mongosh​

转到Atlas集群上的连接选项卡，选择“使用MongoDB Shell连接”选项

下载适用于您操作系统的相关shell并启动连接命令

请参阅以下指南了解更多详细信息。

作为练习，您可以在test数据库下创建一个名为users的集合：

show databases
use test
db.users.insertMany([
    {
      "name": "Alice",
      "age": 26,
      "email": "alice@abc.com"
    },
    {
      "name": "Bob",
      "age": 43,
      "email": "bob@def.com"
    },
    {
      "name": "Carol",
      "age": 19,
      "email": "carol@xyz.com"
    }
])
db.users.find()
